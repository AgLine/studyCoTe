처음에 문제를 앞뒤가 ( ) 로 끝나기만 하면 된다고 이해함
그래서 아래와 같이 코드를 작성햇었음
```
class Solution {
    boolean solution(String s) {
        char firstChar = s.charAt(0);
        char lastChar = s.charAt(s.length() - 1); 
        
        // 첫 글자가 '('가 아니면 바로 실패
        if (firstChar != '(') {
            return false;
        }

        // 마지막 글자가 ')'가 아니면 바로 실패
        if (lastChar != ')') {
            return false;
        }

        //둘다 아니면 성공
        return true; 
    }
}
```
테스트코드는 통과했지만 정답이 아니었음

문제를 다시 이해한후 아래와 같이 코드를 작성함
```
class Solution {
    boolean solution(String s) {

        int cnt = 0;
        for(int i = 0; i < s.length(); i++){
            if(s.charAt(i) == '('){
                cnt++;
            }else{
                cnt--;
            }

            //cnt가 음수가 되면 닫힘이 먼저온것이기 때문에 바로 fasle
            if(cnt < 0){
                return false;
            }
        }

        //cnt의 합이 0이어야 괄호의 짝이 맞는것임
        if(cnt == 0){
            return true;
        }

        return false;
    }
}
```

이후 다른사람의 풀이를 봤는데 스택을 활용하여 풀이를 한 코드를 보게됨

```
import java.util.Stack;

class Solution {
    boolean solution(String s) {
        // Character 타입을 담는 스택 생성
        Stack<Character> stack = new Stack<>();

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);

            if (c == '(') {
                // 열린 괄호면 스택에 넣습니다.
                stack.push(c);
            } else {
                // 닫힌 괄호일 때
                // 만약 스택이 비어있다면? 짝꿍(열린 괄호)이 없으므로 바로 false
                if (stack.isEmpty()) {
                    return false;
                }
                // 짝이 맞으므로 스택에서 열린 괄호 하나를 제거합니다.
                stack.pop();
            }
        }

        // 모든 루프가 끝난 뒤 스택이 비어있어야 모든 짝이 맞은 것입니다.
        return stack.isEmpty();
    }
}
```
